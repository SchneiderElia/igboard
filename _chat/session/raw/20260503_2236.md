- Rimosso il fallback automatico per favorire la trasparenza degli errori verso l'utente.

### 5. Tiering & UI
- Allineata la lista modelli alla Home Page.

- Aggiunto "Usage Monitor" (RPM) nell'header della v3.
- Configurato `VersionSwitcher` globale per nascondersi automaticamente nella rotta v3.

## Decisioni Prese
- **Isolamento v3**: Tutto il codice v3 deve risiedere nelle cartelle dedicate per evitare regressioni sulle versioni precedenti.
- **Formato Fisso**: La tela v3 lavora esclusivamente in 1080x1350 per standardizzare l'output social.
- **Trasparenza Errori**: Preferito mostrare errori specifici (Quota/Billing) invece di fallback silenziosi per una migliore diagnostica.
- **Layout Persistente**: Il Canvas rimane sempre aperto in split-screen per favorire il workflow di iterazione continua sul design.
